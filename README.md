# RobustAction

Padgett & Ansell (1993) "Robust Action and the Rise of the Medici, 1400-1434" を読んで、公開データセットを使ってRで遊んでみたもの。

`ergm` パッケージの組み込みデータ `florentine` を使い、婚姻・ビジネスネットワークの中心性分析、ノード除去シミュレーション、コミュニティ検出などを行っている。
